Recap

  1. use Boots *-starter parents in your pom.xml
  2. start with a @SpringBootApplication-annotated main-class
  3. use @Configuration-annotated classes to tweak your system
  4. get familiar with @Enable*-annotations
  5. define a bunch of @Component-classes as building blocks

When having a whole bunch of microservices how to deal with ...

  • ... service-discovery : Eureka
  • ... reciliency : Hystrix (Circuit Breaker)
  • ... configuration-management : Cloud-Config
  • ... monitoring : Zipkin (Distributed Tracing)
